To build this project locally:

1.  **Clone the repository**
    ```bash
    git clone https://github.com/wuyan1337/SentralNextgen.git
    ```
2.  **Open in Android Studio**
    *   File > Open > Select the cloned directory.
3.  **Sync & Build**
    *   Wait for Gradle sync to complete.
    *   Run `Run > Run 'app'` or use `./gradlew assembleDebug`.

## üîê Privacy & Security

We take your privacy seriously:
*   **No Tracking**: We do not collect usage data, location, or analytics.
*   **Local Storage**: Your credentials and cookies are stored **only** on your device using Android's secure EncryptedSharedPreferences.
*   **Direct Connection**: The app communicates directly with your school's Sentral server. No middleman servers are involved.
